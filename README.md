# Data-Preprocessing-and-ML-Full-Pipeline

This repository provides a complete end-to-end workflow for data science projects, ranging from raw data preprocessing and feature engineering to advanced machine learning modeling using both regression and classification techniques.


## 📁 Project Structure
* **Notebooks:** `.ipynb` files containing step-by-step implementations.
* **Data:** `.csv` files used for training and testing.
* **Visualizations:** Seaborn and Matplotlib plots for insights.

## 🛠 Tech Stack
* **Languages:** Python
* **Visualization:** Matplotlib, Seaborn
* **Preprocessing:** Scikit-learn (Feature Engineering, Encoding, Resampling)
* **Models:** Scikit-learn, XGBoost, LightGBM

---

## 🔍 Content Overview

### 1. Data Visualization & Preprocessing
* **Seaborn & Matplotlib:** Advanced plotting (Correlation heatmaps, distribution plots, box plots).
* **Feature Engineering:** Handling missing values, outlier detection, and feature creation.
* **Encoding & Resampling:** One-Hot/Label Encoding and addressing class imbalance via resampling techniques.

### 2. Regression Analysis
* **Linear Regression:** Basic trend prediction.
* **Polynomial Regression:** Modeling non-linear relationships.
* **Logistic Regression:** Fundamental classification approach.

### 3. Supervised Learning Algorithms
* **K-Nearest Neighbors (KNN):** Distance-based classification and regression.
* **Naive Bayes:** Probabilistic modeling.
* **Support Vector Machines (SVM):** Kernel-based classification.

### 4. Tree-Based Models & Ensemble Learning
Comprehensive study of Regressor and Classifier variants for:

* **Decision Trees:** The foundation of tree-based logic.
* **Random Forest:** Bagging technique for variance reduction.
* **AdaBoost & Gradient Boosting:** Sequential error correction.
* **XGBoost & LightGBM:** High-performance, scalable boosting frameworks.

## 🔍 İçerik Özeti

### 1. Veri Görselleştirme ve Ön İşleme
* **Seaborn & Matplotlib:** Gelişmiş grafikler (Korelasyon ısı haritaları, dağılım grafikleri, kutu grafikleri).
* **Özellik Mühendisliği (Feature Engineering):** Eksik veri yönetimi, aykırı değer tespiti ve yeni özellik türetme.
* **Encoding & Resampling:** One-Hot/Label Encoding ve yeniden örnekleme teknikleri ile sınıf dengesizliğinin giderilmesi.

### 2. Regresyon Analizi
* **Lineer Regresyon:** Temel trend tahmini.
* **Polinomal Regresyon:** Doğrusal olmayan ilişkilerin modellenmesi.
* **Lojistik Regresyon:** Temel sınıflandırma yaklaşımı.

### 3. Denetimli Öğrenme Algoritmaları
* **K-En Yakın Komşu (KNN):** Mesafe tabanlı sınıflandırma ve regresyon.
* **Naive Bayes:** Olasılık tabanlı modelleme.
* **Destek Vektör Makineleri (SVM):** Çekirdek (kernel) tabanlı sınıflandırma.

### 4. Ağaç Tabanlı Modeller ve Topluluk (Ensemble) Öğrenmesi
Aşağıdaki modellerin hem **Regressor** (Regresyon) hem de **Classifier** (Sınıflandırıcı) varyantlarının kapsamlı çalışması:
* **Karar Ağaçları:** Ağaç tabanlı mantığın temeli.
* **Random Forest:** Varyansı azaltmak için Bagging tekniği.
* **AdaBoost & Gradient Boosting:** Ardışık hata düzeltme yöntemleri.
* **XGBoost & LightGBM:** Yüksek performanslı ve ölçeklenebilir boosting kütüphaneleri.
